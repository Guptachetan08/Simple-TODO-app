
# Simple TODO app

This is a TODO app that can be used to maintain a TODO lsit of tasks in browser tab.
But as it is not having any database so the list vanishes off as soon as the browser window or tab is closed.




## Features

- Add tasks
- Reset Button
- Sort the tasks
- Clean up the list



## Tech

- HTML
- CSS
- JavaScript
